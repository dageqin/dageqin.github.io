##　什么是angular

  • 学习曲线-难（入门容易）

  • MVVM/MVC
    - mvc
    - model(数据) view（视图） controller（控制器）
    - view（视图）人为操纵-->controller（控制器）-->model-->view

使用angular

安装angular

npm node package manager

  • 先初始化package.json文件

  • 安装angularjs

在当前的文件夹下生成node_moduls文件将安好的内容放入（package.json中的name不能叫angular）

使用angular

  • ng-app
    让angular自行启动，并且创建$rootScope跟作用域

  • ng-model (表单元素上) 只能放变量名，不能有任何运算
    实现双向数据绑定

    如果ng-model用点的形式 取得是点前面对象的属性，没有的话，就会声明

  • ng-init
    初始化数据

  • 取值表达式
    取当前作用域上的值，只支持运算、赋值、三元表达式

    防止闪烁

  • ng-bind （适合单一元素）绑定 属性用户是看不到的，将angular编译好后的数据html到div中，标签内不能放置数据 适合单一元素
  • ng-cloak （适合多个元素）需要用户自己隐藏，angular加载后会移除掉ng-cloak属性

  • 生成模块

  • 生成控制器

  • 应用控制器

控制器的特点

  • 1.主要控制数据
  • 2.不要在控制器中操作DOM（link函数中）
  • 3.控制的作用范围（与DOM标签结构平级）
  • 4.控制器可以嵌套（嵌套的是DOM节点，不是控制器）
  • 5.有继承关系，父作用域不能继承子作用域，但是子作用域可以继承父作用域
